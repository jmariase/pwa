How to Connect to Your Port.io Workspace

Complete configuration files are generated in this repository. Follow these steps to map them:

1

Create Blueprints in Port.io

Go to your Port Builder and import the blueprint definitions found in the file .port/blueprints.json.

2

Configure Self-Service Action

Add the deploy action to Port.io using the definition in .port/actions.json to trigger GitHub deployments.

3

Configure GitHub Secrets

Add PORT_CLIENT_ID, PORT_CLIENT_SECRET, and SONAR_TOKEN to your GitHub Repo Secrets.
